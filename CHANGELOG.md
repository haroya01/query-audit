# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- 5 new detection rules: LIMIT without ORDER BY, Window function without PARTITION BY, FOR UPDATE without NOWAIT/SKIP LOCKED, CASE in WHERE, FORCE/USE/IGNORE INDEX hints (total: 64 rules)
- Per-rule enable/disable via `disabled-rules` configuration
- Per-rule severity override via `severity-overrides` configuration
- Configurable thresholds for 7 additional rules: large-in-list, too-many-joins, excessive-column, repeated-insert, write-amplification, slow-query (warning/error)
- Spring Boot application.yml support for all new configuration options
- JSqlParser fallback logging at DEBUG level

### Fixed
- Dual normalization bug: QueryRecord now delegates to SqlParser.normalize() for consistent normalization
- MergeableQueriesDetector regex backtracking vulnerability (removed Pattern.DOTALL)
- CaseInWhereDetector: removed unused static fields, cached Pattern compilation
- datasource-proxy dependency scope changed from `api` to `implementation`
